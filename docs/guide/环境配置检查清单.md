## 🚨 常见问题解决

### 1. Java环境问题
**症状**：`java -version` 不显示或版本不对
**解决**：
1. 重新下载JDK安装包
2. 正确设置JAVA_HOME环境变量
3. 重启命令行或IDE

### 2. MySQL连接失败
**症状**：无法连接MySQL，提示"Access denied"
**解决**：
```sql
-- 重置root密码（MySQL 8.0）
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码';
FLUSH PRIVILEGES;
```

### 3. Redis服务无法启动
**症状**：Redis启动失败，端口被占用
**解决**：
```bash
# 查找占用6379端口的进程
netstat -ano | findstr :6379  # Windows
lsof -i :6379                 # Linux/Mac

# 停止占用进程或修改Redis配置文件端口
```
